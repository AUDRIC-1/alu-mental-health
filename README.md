 ALU Mental Health Support Application

A safe and private platform for ALU students to manage their mental health.

Live App
https://audric.netlify.app

About the Project
Many ALU students struggle with mental health challenges but don't have easy access to support. This platform gives students a private and safe space to check in on their mental health, access self-help resources, and connect with counselors.

Features
- **Register & Login** – Students create a secure account
- **Mental Health Check-in** – Students rate their mood and get instant feedback
- **Self-Help Resources** – A library of helpful articles and guides
- **Counseling Booking** – Students can request a session with a counselor

Tech Stack
- **Frontend:** React
- **Backend:** Node.js + Express
- **Database:** JSON file storage

How to Run Locally

What you need installed
- Node.js (https://nodejs.org)
- Git (https://git-scm.com)

Step 1 - Clone the project
```
git clone https://github.com/AUDRIC-1/alu-mental-health.git
cd alu-mental-health
```

Step 2 - Start the Backend
```
cd backend
npm install
node index.js
```
Backend will run on http://localhost:5000

Step 3 - Start the Frontend
Open a new terminal and run:
```
cd frontend
npm install
npm start
```
App will open on http://localhost:3000
